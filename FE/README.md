### Installation Steps

1. **Install dependencies**
```bash
   npm install
```

2. **Start the app**
```bash
   npm run dev
```

3. **Open in browser**
   - Navigate to `http://localhost:5173`

## Tech Stack
- **Vue.js 3**
- **Pinia**
- **Vite**
